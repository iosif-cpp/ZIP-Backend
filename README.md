# Backend для архивирования файлов

## Описание

- Rest API сервис предоставляющий функционал по созданию zip архивов
- По пути \Kaspersky-Task1\files находится директория, в которой вы можете сохранять файлы для дальнейшего архивирования

## Эндпоинты

- **POST** `/api/archives/init` - инициализация процесса архивирования коллекции файлов
- **GET** `/api/archives/{processId}/status` - проверить статуст процесса
- **GET** `/api/archives/{processId}/download` - получить поток для скачивания файла
- **GET** `/api/files` - получить список всех файлов в директории \files


## Как запустить?


0) Убедитесь что порт 5278 никто не слушает(вы можете в appsettings.json указать удобный вам порт)
1) В корне проекта  выполнить "docker compose up --build"
2) После запуска вы можете перейти по адрессу "http://localhost:5278/swagger/index.html"


## Как использовать?

1) Добавьте в \files ваши файлы
2) Используйте эндпоинт init который отдасть вам id процесса по работе архивирования
3) Используйте эндпоинт status чтобы убедиться, что нужный вам процесс в статусе done
4) Через эндпоинт download укажите id и скачайте архив
